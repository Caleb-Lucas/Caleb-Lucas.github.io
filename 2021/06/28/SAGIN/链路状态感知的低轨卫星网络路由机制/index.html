链路状态感知的低轨卫星网络路由机制

发表于 2021-06-28 分类于 SAGIN

论文引用

论文编号 论文名称
7 Li T,Zhou H,Luo H,et al.Using SDN and NFV to implement satellite communication networks [C] 2016 International Conference on Networking and Network Applications (NaNA),IEEE

概述

  • 卫星网络作为中继节点，在地面站之间进行通信。与地面网络相比，卫星网络具有全天候、高带宽、覆盖范围广、不受地形限制等优点。高效的卫星网络路由方案对提高卫星网络传输的时效性和可靠性有着重要意义。与其他类型的卫星网络相比，低轨卫星网络既具有传输耗损小、通信延迟低、功耗低等优点，也存在拓扑变化快、卫星节点的计算能力和功率受限和易出现拥塞等特点。
  • 现有研究大多数关注卫星网络拓扑结构的动态变化，提出了基于系统周期分割思想和覆盖区域分割思想，将卫星网络结构由动态转化为静态，再在静态结构下进行协议设计进行路由的方法。其中系统周期分割思想根据卫星网络的周期性，将卫星系统周期划分为多个时间片。只要时间片足够小，就可看作卫星网络拓扑结构在同一个时间片内是不变的。于是卫星网络就被视为周期性重复的一系列拓扑快照，卫星网络路由计算也就转换为多个静态拓扑下的路由计算问题。基于系统周期分割思想的卫星网络，卫星不需要实时计算，协议开销较小。但是系统周期性分割思想需要大量的空间存储拓扑快照，并对流量变化、链路拥塞及故障等实时情况的适应性较差。覆盖区域划分思想将地球表面规则地划分为不同的区域，给每个节点赋予不同的固定逻辑地址。在覆盖区域划为思想下，逻辑节点的位置固定且排列规则，当前很多研究工作将集中控制与覆盖区域思想相结合，实现流量传输的最优控制。但是覆盖区域分割思想需要卫星拓扑结构非常规则，一旦某个卫星节点出现故障，网络拓扑规则性被破坏，就会导致卫星网络性能大打折扣。另外，在覆盖区域分割思想下，卫星节点需要实时进行路由计算，对卫星星上处理能力要求高。

机制原理

  • 感知驱动路由机制的实现由数据包转发模块和路由学习模块组成,路由学习模块通过SRLM包的广播实现，其中源节点的信息包括卫星ID、剩余能量、接收队列长度等。在感知驱动路由机制中，每个卫星节点都会周期性地广播SRLM报文，并转发来自其他节点的SRLM报文。SRLM报文的广播使得节点信息在全网范围内传输。节点接收到新的来自其他节点的SRLM报文时，根据报文信息更新路由表。路由决策是指节点依据当前的路由信息对数据包进行路由。
  • 路由学习过程指节点通过SRLM报文感知链路质量、邻居节点的接收队列长度和剩余能量信息的过程。其中，邻居节点的接收队列长度和剩余能量长度可通过邻居节点生成的学习包获得。链路质量（TQ）则通过SRLM报文的丢包率计算得到。通过对链路质量进行单跳计算和多跳计算，得到完整一条链路的质量值
  • 路由决策：在保证数据传输质量的情况下，为了减少卫星节点能量消耗并改善卫星网络节点局部拥塞的情况，链路状态感知路由机制通过对链路质量、拥塞情况以及剩余能量进行权衡选择下一跳节点。当从卫星节点m发送数据包至卫星节点n时，卫星节点通过式（6）、式（7）进行路由决策
